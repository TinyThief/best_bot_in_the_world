# Аналитический блок бота: что есть и чего не хватает

## Что уже есть (достаточно для решений)

| Блок | Что даёт | Где используется |
|------|----------|------------------|
| **Тренд** | Направление (up/down/flat), сила, режим (trend/range/surge), уверенность | Фильтр входа, сигнал по старшему ТФ, лог |
| **Фаза** | 6 фаз (accumulation → recovery), score, неясность, устойчивость, RSI и дивергенция внутри фазы | Фильтр phase_decision_ready, сигнал long/short/none, лог |
| **Зоны** | Уровни S/R с переворотом ролей, объём на уровне, зона ±ATR, круглые числа, свежесть, composite strength, at_support_zone / at_resistance_zone, recent_flips | Расстояние до уровней (фильтр level_ok), лог, контекст «где цена» |

**Связка:** multi_tf агрегирует тренд и фазу по ТФ, фильтрует по объёму/ATR/уровням/режиму/качеству свечей, выдаёт direction (long/short/none), entry_score, confidence. Зоны дают «у какого уровня мы находимся» и «какие уровни недавно перевернулись».

Для **принятия решения «входить или нет» и «в какую сторону»** этого достаточно: тренд + фаза + зоны + фильтры + единый score.

---

## Чего не хватает (опционально)

Не обязательны для рабочего бота, но могут усилить аналитику.

| Дополнение | Зачем | Сложность |
|------------|--------|-----------|
| **Явный momentum state** | Один признак: «импульс сильный / затухает / нейтральный» (например, по RSI или return + объём). Сейчас RSI и return есть внутри фаз и тренда, но отдельного «momentum» в отчёте нет. | Низкая |
| **Конfluence зон по ТФ** | Один и тот же уровень на 15m, 1h, 4h — повышать силу уровня. Уже в плане (TRADING_ZONES_PROP_UPGRADE). | Средняя |
| **Подтверждение пробоя** | Переключать роль уровня только после ретеста или всплеска объёма на пробое — меньше ложных переворотов. | Средняя |
| **Сессии (Asian/London/NY)** | Контекст «сейчас какая сессия» для фильтра или логирования. Нужно только если стратегия завязана на сессии. | Низкая |
| **Ликвидность (equal highs/lows, sweep)** | Явные зоны ликвидности и «свип и отбой». Для продвинутого сценария. | Выше средней |

## Итог

- **Достаточно ли блока?** **Да.** Тренд + фаза + зоны покрывают: направление, режим рынка, стадию цикла и расположение цены относительно уровней. Вместе с фильтрами и entry_score этого хватает для аналитического ядра без исполнения сделок.
- **Что добавить в первую очередь (если захочется усилить):**
  1. **Явный momentum** — один признак в отчёт (например, RSI или «импульс затухает») для фильтра «не входить при выдохшемся импульсе».
  2. **Конfluence зон по ТФ** — один уровень на нескольких ТФ = сильнее.
  3. **Подтверждение пробоя** — меньше ложных переворотов ролей уровней.

Остальное (сессии, ликвидность) — по мере появления стратегий, которые этого требуют.
