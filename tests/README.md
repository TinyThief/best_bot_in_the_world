# Тесты

- **unit/** — юнит-тесты (config, exchange, strategy, risk и т.д.). Мокировать биржу и БД.
- **integration/** — интеграционные тесты (полный поток: БД, биржа, анализ).
- **backtest/** — тесты и скрипты бэктеста по историческим данным.

Быстрая проверка окружения и импортов: **python check_all.py** из корня проекта.
