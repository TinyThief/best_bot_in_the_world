# Стратегии

Здесь будут отдельные модули стратегий (trend_following, scalping, mean_reversion и т.д.).

Сейчас логика сигнала живёт в **src/analysis/multi_tf.py** и **src/analysis/market_phases.py**: тренд по старшему ТФ, 6 фаз рынка, direction long/short/none.

При добавлении новых стратегий:

- наследовать от абстрактного базового класса (если введёшь его в core или здесь);
- реализовать расчёт сигнала и валидацию;
- не хардкодить пороги — брать из конфига.
